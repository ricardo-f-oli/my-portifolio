The Vendors API provides a complete solution for managing vendor data within enterprise systems. Built with modern Java technologies and following REST best practices, this API offers seamless integration capabilities with comprehensive error handling, validation, and monitoring.

## ðŸš¦ Quick Start Guide

### 1. Clone and Prepare the Repository

```bash
# Clone the repository
git clone https://github.com/your-username/vendors-api.git
cd vendors-api

# Install dependencies and verify build
mvn clean install
```

### 2. Start Services with Docker

```bash
# Start all required services
docker-compose up -d
```

This command initializes:

* PostgreSQL database
* External API dependencies
* Prometheus for metrics collection
* Grafana for visualization dashboard

### 3. Run the Application

```bash
# Run in development mode
mvn spring-boot:run

# Or run with debugging enabled
mvn spring-boot:run -Dspring-boot.run.jvmArguments="-Xdebug -Xrunjdwp:transport=dt_socket,server=y,suspend=n,address=5005"
```

The API will be available at:

```
http://localhost:8080
```

### Update Existing Vendor

**PUT** `/{id}`

**Path Parameter**

* `id`: Vendor ID to update (string)

**Request Body**

```json
{
  "name": "John Updated",
  "birthday": "01-01-2017",
  "document": "CPF ou CNPJ",
  "email": "updated@email.com",
  "contractType": "clt",
  "branch": "updated branch name"
}
```
